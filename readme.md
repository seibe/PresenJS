# Speech
Notice: This project is work in progress.

Speechは、MeetupにおけるプレゼンテーションをWEB上でリアルタイムに配信・記録できるシステムです。

## How to use
現在、Windows / Mac OS X上で動作を確認しています。
利用のためのドキュメントは2016年度中を目途に提供予定です。
